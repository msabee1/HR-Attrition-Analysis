# HR Attrition Analysis Dashboard 📊

## 📌 Overview

This project focuses on analyzing employee attrition using an interactive dashboard built in Power BI. It helps identify key factors influencing employee turnover and supports better HR decision-making.

## 🛠 Tool Used

* Power BI

## 📂 Dataset Features

* Age
* Job Role
* Salary
* Years at Company
* Attrition (Yes/No)

## 📊 Dashboard Features

* Attrition rate overview
* Job role-wise attrition analysis
* Salary vs attrition comparison
* Age group distribution
* Years at company analysis
* Interactive filters (slicers) for detailed exploration

## 📈 Key Insights

* Higher attrition observed among employees with lower salaries
* Certain job roles have higher turnover rates
* Employees with fewer years at the company are more likely to leave
